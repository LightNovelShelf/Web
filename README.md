# 轻书架 (light_novel_shelf)

A PWA-focused online novel reading project built with Quasar.

## Install dependencies

```bash
npm install
```

## Start development

```bash
npm run dev
```

## Lint

```bash
npm run lint
```

## Format

```bash
npm run format
```

## Build

```bash
npm run build
```

## Build PWA

```bash
npm run build:pwa
```

## Configuration

See the Quasar config documentation for customization details.
